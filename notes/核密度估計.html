密度估計是由觀測數據估計母體的概率密度函數。參數估計法是數據源自已知母體分布，運用似然函數估計母體分布之參數。非參數估計法是數據源自未知母體分布，估計法包含直方圖、羅森布來特估計。

設隨機變數X之母體機率函數為F，密度函數為f，則有\(F(b)-F(a)=P\{a \le X \le b\}=\int_a^b f(u)du\)。從中抽出n個之樣本\(\{X_1, X_2,…,X_n\}\)，則樣本落入區間[a, b]的個數為\(\sum_{i=1}^{n} 1\{a \le X_i \le b\}\)。次依大數定律知道當n足夠大時，\(F(b)-F(a)=P\{a \le X \le b\}=\frac{\sum_{i=1}^{n} 1\{a \le X_i \le b\}}{n}\) 則\(f(b-a) = \frac{dP(b-a)}{d(b-a)} = \frac{\sum_{i=1}^{n} 1\{a \le X_i \le b\}}{n(b-a)}\)

設\(f\)為隨機變數\(X\)之密度函數，則有取起點\(x_0\)及帶寬h將x軸分成無限個\([x_0+mh, x_0+(m+1)h]\)區間。

直方圖缺點非連續函數。
